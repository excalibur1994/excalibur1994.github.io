Docker是什么

Docker可以减少服务器环境的配置，类似于一个镜像系统。我们在每次建linux服务器的时候，都需要配置MySQL，Tomcat之类的服务器环境，而使用Docker可以将这些配置封装成一个镜像，当有新的服务器需要配置时，我们在新的服务器中直接使用这个镜像，就可以省去很多相同的配置。

Docker核心概念

docker主机（Host）：安装了Docker程序的机器（Docker直接安装在操作系统之上）

docker客户端（Client）：链接Docker主机进行操作

docker仓库（Registry）：用来保存各种打包好的软件镜像

docker镜像（Images）：将软件打包后形成镜像

docker容器（Container）：镜像启动好后的实例称为一个容器，容器是独立运行的一个或一组应用

使用Docker的步骤

  1. 安装Docker
  2. 去Docker仓库找到这个软件对应的镜像
  3. 使用Docker运行这个镜像，这个镜像就会生成一个Docker容器
  4. 对容器的启动停止就是对软件的启动停止

安装Docker

安装前提：Docker要求CentOS系统的内核版本高于3.10，可以使用uname -r查看内核版本。如果版本过低可以使用yum update进行升级。

下载docker：yum install docker，安y确认安装。

启动docker：systemctl start docker，使用docker -v可以查看版本号，如果出现版本号说明安装成功。

自动启动docker：systemctl enable docker。

关闭docker：systemctl stop docker。

常用操作

镜像操作

检索：docker search 关键字。（例如 docker search mysql）

拉取：docker pull 镜像名:tag（tag多表示软件的版本,具体的tag分类可以前往docker hub查看，如果不加则默认下载最新的版本，例如想要下载MySQL5.5，就可以输入docker pull mysql:5.5）。

列表：docker images 查看所有本地镜像。

删除：docker rmi image-id删除指定的本地镜像，image-id是每个镜像唯一的id，查看本地所有镜像时就可以看到。
